RIFT: On-Chain Dark Forest

A persistent strategy game entirely powered by Somnia Native Reactivity. No servers, no off-chain game masters, no keepers.

01 — The Problem

Web3 games aren't really on-chain.

Most blockchain games rely on centralized servers or off-chain bot "keepers" to manage game states, process ticks, and handle logic.

❌ Off-Chain Processing

If the game server dies, the game dies. Actions are permissioned by the developer's infrastructure.

❌ Isolated Ecosystems

The game state is completely ignored by the rest of the blockchain's TVL and DeFi action.

02 — The Solution

The blockchain is the Game Master.

🌐

Native Reactivity

Built with Somnia's Reactive SDK. The game Mutates in the same block that external events happen.

💥

Dynamic External Triggers

Mock DeFi Liquidations, Price Crashes, and DAO Votes directly alter the 49-hex physical map.

🕹️

Live HD-2D UX

A fully decentralized glassmorphic, pixel-art tactical frontend reading strictly from on-chain RPC calls.

03 — Architecture

5 Custom Reactive Watchers.

Price Oracle Watcher

Listens for 3% price drops. Triggers an Economic Shockwave, destroying market zone wealth.

Lending Protocol Watcher

Listens for liquidations. Triggers Territory Collapses on the borders, opening them to attack.

ERC20 Whale Watcher

Listens for massive transfers. Triggers a Seismic Event that scrambles fault lines.

DAO Governance Watcher

Listens for proposals. Generates Relics in Oracle Zones to grant buffs.

04 — Deployment

14 Contracts LIVE on Somnia Testnet

The `RiftWorld` engine, player registrations, mock DeFi protocols, and all 5 Reactivity Watchers are live on the Shannon Testnet.
